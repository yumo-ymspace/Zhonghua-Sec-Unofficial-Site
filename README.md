📖 About The Project

Traditional school websites often suffer from information silos, burying critical daily information like class timetables and Co-Curricular Activity (CCA) schedules inside static PDF documents.

This project aims to solve that friction by creating a Task-Oriented Web Architecture. Built entirely without heavy frontend frameworks (like React or Angular), this site leverages Vanilla JavaScript and Tailwind CSS to deliver a lightning-fast, accessible, and mobile-first experience.

It serves as a centralized hub connecting students to vital school services, official resources, and student-led initiatives like ZNotes and ZPortal.

✨ Key Features

📅 Smart Timetable Hub

Replaces the need to hunt through PDF directories. Students can select their stream and class from a dropdown to instantly generate a direct link to their official semester schedule.

🏀 Dynamic CCA Explorer

An interactive, filterable grid of all CCAs. Users can easily sort by Sports, Performing Arts, Uniformed Groups, or Clubs, and view specific training times and objectives via smooth modal popups.

🏆 Achievements Visualizer

Utilizes Chart.js to render interactive Radar and Bar charts, visualizing the school's 5 Pillars of Best Practice and historical academic performance trends.

🔗 Student App Integrations

Dedicated landing pages and seamless redirects to essential student utilities like the ZNotes collaborative sharing platform and the ZPortal personalized dashboard.

📱 Mobile-First UI/UX

Features a sticky navigation bar, smooth slide-down mobile menus, page-load fade animations (page-enter), and tailored responsive grids inspired by modern Bento Grid design.

🎨 "Begonia" Design System

A custom color palette utilizing Emerald and Slate tones, honoring the school's heritage (Green and White) while strictly adhering to modern WCAG accessibility contrast standards.

🛠️ Built With

This project is intentionally built "Vanilla" to ensure long-term sustainability, maximum performance, and easy maintainability.

Structure: Semantic HTML5

Styling: Tailwind CSS (via CDN)

Interactivity: Vanilla JavaScript (ES6+)

Data Visualization: Chart.js

Typography: Inter (Google Fonts)

🗂️ Repository Structure

The site is built across several static HTML files, unified by consistent header and footer components:

- index.html

The Home dashboard featuring school announcements and quick links.

- about.html

History, Principal's message, core identity (Vision/Mission/RICE values), and school song.

- timetable.html

The interactive class schedule generator.

- cca.html

The filterable CCA database and schedules.

- achievements.html

School awards and data visualizations (Chart.js).

- znotes.html

Landing page redirecting to the collaborative ZNotes platform.

- zportal.html

Landing page redirecting to the centralized ZPortal student dashboard.

- 404.html

Custom styled "Page Not Found" error page.

- 502.html

Custom styled "Bad Gateway" error page.
